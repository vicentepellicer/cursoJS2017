eval(string)

Función que evalua una expresión representada por el string

                var suma=eval("2+3");
                alert("La suma de 2 + 3 es "+suma);
                var x=5;
                var y=3;
                alert("La multiplicación de "+x+" * "+y+" es "+eval("x*y"));
