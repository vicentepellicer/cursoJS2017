Do While

La sentencia do...while se repite hasta que una condición especificada se evalúa a false

                  do {
                    i += 1;
                    console.log(i);
                  } while (i < 5);

While

Una sentencia while ejecuta sus sentencias mientras la condición sea evaluada como verdadera 

                var n = 0;
                var x = 0;
                while (n < 3) {
                  n++;
                  x += n;
                  console.log("El valor de n es "+n);
                  console.log("El valor de x es "+x);
                }

For

Un bucle for se repite hasta que la condición especificada se evalúa como false 

                for (var i = 0; i < 10; i++) {
                  console.log("El valor de i es "+i);
                }
