var

La sentencia var declara una variable, opcionalmente inicializándola con un valor.

                variable=3;
                var variable;
                alert("El valor de variable es: "+variable);

let

La sentencia let declara una variable de alcance local, la cual, opcionalmente, puede ser inicializada con algún valor.

                for (let i = 0; i<10; i++) {
                  alert("Variable local i: "+i); // 0, 1, 2, 3, 4 ... 9
                }
                alert("Variable fuera de su ambito: "+i); // i is not defined

const

as constantes presentan un ámbito de bloque tal y como lo hacen las variables definidas usando la instrucción let, con la particularidad de que el valor de una constante no puede cambiarse a través de la reasignación, y no se puede redeclarar.

                const PI = 3.14;
                PI = 3;
                alert(PI);
